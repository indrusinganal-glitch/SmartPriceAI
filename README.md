# SmartPrice AI

SmartPrice AI is a Django-based smart price comparison web application.

## Features
- User Registration and Login
- Product Dashboard
- Product Search
- Category Filter
- Product Details
- Multi-platform Price Comparison
- Wishlist
- Price History Graph
- Price Drop Alerts
- AI-based Recommendations
- Product Ratings and Reviews
- Admin Analytics Dashboard
- Dark Mode

## Tech Stack
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
- Chart.js
